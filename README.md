# cutetons

> Made with create-react-library

[![NPM](https://img.shields.io/npm/v/cutetons.svg)](https://www.npmjs.com/package/cutetons) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save cutetons
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'cutetons'
import 'cutetons/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
